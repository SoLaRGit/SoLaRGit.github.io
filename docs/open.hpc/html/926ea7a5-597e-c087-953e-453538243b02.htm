Starts tracking of the HPCMeter calling Track(HPCMeter) method with specified meter, and starts measurement by calling Reset method before return.
Reset(String)

ShowStatistics
Returns composed performance measurements report of cumulative (Sum) statistics time usage of meters tracked by this instance.
 Note

■ NOTE: This method is not optimized, and do not use it when measuring performance. Especially do not output it to debug window while measuring. It is meant to be used after all measurements are done.

Track(HPCMeter)

Adds meter to TrackedMeters collection if not already added.

If not already added it resets it's internal sum data to initial state, then meter is added to TrackedMeters collection.

Track(String)
Tracks a HPCMeter by meter name with this collection.

If a HPCMeter instance with specified meterName exists in collection, it shall be used.

Otherwise, if meter does not exist, a new instance of HPCMeter class shall be created with meterName, and added to the collection.

 Note

■ NOTE: Before this method exits meter sum data is reset by calling Reset method.
