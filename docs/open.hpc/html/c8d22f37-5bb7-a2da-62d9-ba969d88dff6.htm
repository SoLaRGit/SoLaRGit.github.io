CurrentTime
Gets current high performance counter value, a time in Frequency units, and can be used for progress evaluation purposes.
 Note

■ NOTE: Sets StopTime to current value of high performance counter. But does not set StartTime member values, and uses previously recorded values to return duration.

SumAverage

Returns the sum average value, if SumCount is greater from zero : (SumTime / SumCount);

Otherwise returns zero.
