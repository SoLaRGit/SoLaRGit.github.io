Returns composed performance measurements report of cumulative (Sum) statistics time usage of meters tracked by this instance.
 Note

■ NOTE: This method is not optimized, and do not use it when measuring performance. Especially do not output it to debug window while measuring. It is meant to be used after all measurements are done.
