HPCMeter Methods

The HPCMeter type exposes the following members.

Methods
  NameDescription
GetDuration
Returns duration : ((CurrentTime - StartTime) / Frequency).
 Note

■ NOTE: Sets StopTime to current value of high performance counter. But does not set StartTime member values, and uses previously recorded values to return duration.

GetLastDuration
Returns last duration : ((StopTime - StartTime) / Frequency).
 Note

■ NOTE: Does not set StopTime or StartTime member values, but uses previously recorded values to return duration.

Reset
Sets StartTime to current value of performance counter.
 Note

■ NOTE: Begins the new measurement interval.

Reset(String)
Sets current instance Name member to newMeterName value, and calls Reset method to start measurement.
 Note

■ NOTE: Begins the new measurement interval.

ResetSumData
Resets sum statistics data members as if there were no measurements at all for this meter, and starts new measurement interval.
 Note

■ NOTE: The following members are reset to defaults:

SumTime, SumCount, SumTimeMin, SumTimeMax, SumTimeMinOccurrence, and SumTimeMaxOccurrence.

 Note

■ NOTE: After reseting sum statistical fields, method Reset is called.

Sum
Sets StopTime to current value of high performance counters, and ends measurement interval by collecting the sums statistics data. And returns the SumTime value.
 Note

■ NOTE: The following members are updated with sum statistics data:

SumTime, SumCount, SumTimeMin, SumTimeMax, SumTimeMaxOccurrence, and SumTimeMinOccurrence.
