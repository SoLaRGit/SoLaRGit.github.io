HPCMeterSum

A HPCMeterSum class is used in tracking single HPCMeter, and performing the Sum operation on meter once this instance is disposed. Allowing us to meter the context of `using` code block.

Examples
Copy
using(new HPCMeterSum(varMeter))
{
  /* metered code block */
}
